DM-1on1-300k-lava-pit Unity-compatible export

Files:
- DM-1on1-300k-lava-pit.obj: triangulated map brush geometry
- DM-1on1-300k-lava-pit.mtl: material slots with placeholder colors

Unity import:
1. Drag the whole folder into your Unity project's Assets folder.
2. Select the OBJ and set scale as needed. The mesh is exported in original UT units.
3. Replace placeholder materials with your own textures/materials.

Notes:
- This is a geometry/material-slot export from the UT99 map package.
- It does not convert gameplay actors, bots, pickups, movers, triggers, sounds, pathing, or lighting.
- The source map uses stock UT textures; texture image files were not embedded in the zip, so this export only includes material names/placeholders.
